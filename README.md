# ⚡ File Converter

### A powerful, universal file converter that runs entirely in your browser.<br/>Convert images, documents, and data without ever uploading your files.

<br/>

[![🚀 Live Demo](https://img.shields.io/badge/🚀%20Live%20Demo-file--converter--30in30.vercel.app-8b5cf6?style=for-the-badge)](https://file-converter-30in30.vercel.app/)

## ⚙️ Features

<table>
  <tr>
    <td width="50%">
      <h3>🖼️ Image Converter</h3>
      Convert seamlessly between JPG, PNG, WEBP, and Base64 strings. Includes an adjustable quality slider for JPEG and WebP to compress files locally.
    </td>
    <td width="50%">
      <h3>📊 Data Transformer</h3>
      Instantly parse and convert between JSON, CSV, TSV, and XML formats. Robust parser capable of taking apart structured text to rebuild it in seconds.
    </td>
  </tr>
  <tr>
    <td>
      <h3>📄 Document Generator</h3>
      Create structured PDFs or basic DOC files directly from raw text or JSON structures entirely within the browser client.
    </td>
    <td>
      <h3>🧠 Smart Limitations</h3>
      Intelligently detects unsupported server-side formats (like heavy Videos or precise DOCX) and provides graceful UI handling with clear explanations.
    </td>
  </tr>
</table>

---

## 🔐 Privacy First

> **Your files never leave your device.**
> All files are read and processed directly using your device's memory and the HTML5 FileReader & Canvas APIs.
> No data is ever uploaded to a server or external API for processing.
